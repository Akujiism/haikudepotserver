Paketübersetzungen enthalten Übersetzungen eines Pakets, die sich von Version zu Version nicht unterscheiden. Sie unterscheiden sich von Versionsübersetzungen, die für verschiendene Versionen und Architekturen eines Paket unterschiedlich sein können. Auf der linken Seite befinden sich die verschiedenen Sprachen, an denen gearbeitet wird. Ein Klick auf eine Sprache zeigt die entsprechende Übersetzung. Die verschiedenen Übersetzungen können bearbeitet und mit dem Aktions-Button ganz unten gespeichert werden.
