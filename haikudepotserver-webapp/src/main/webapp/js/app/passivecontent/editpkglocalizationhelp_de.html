Im Fall die Übersetzungen in Paketdatei (die Paketversion) fählt, werden diese Paketübersetzungen als ersatz verwendet. Auf der linken Seite befinden sich die verschiedenen Sprachen, an denen gearbeitet wird. Ein Klick auf eine Sprache zeigt die entsprechende Übersetzung. Die verschiedenen Übersetzungen können bearbeitet und mit dem Aktions-Button ganz unten gespeichert werden. Die Elemente (Title, Kurzfassung und Beschreibung) der Paketversion würde von die folgende Möglichkeiten ausgewählt;

  1. Paketversioneintrag für die Benutzers Sprache
  2. Paketeintrag für die Benutzers Sprache
  3. Paketversioneintrag für die englishe Sprache
  4. Paketeintrag für die englishe Sprache
